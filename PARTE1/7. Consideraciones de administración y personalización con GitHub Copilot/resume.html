Exclusiones de contenido — mnemónico: "CIO: Configura, Impacta, Observa límites"
C — Configuración

Repositorios y organizaciones

↩ rutas
Repositorio:

Repo → Configuración → Copilot → Repositorios y rutas a excluir

Organización:

Org → Configuración → Copilot → Exclusión de contenido

I — Impacto

Qué cambia al excluir un archivo

↩ efectos

❌ No hay autocompletado en archivos excluidos.

❌ El contenido excluido no informa sugerencias en otros archivos.

❌ No informa respuestas de Chat.

⚠️ Puede reducir calidad de sugerencias por menos contexto.

O — Observa límites

Limitaciones de las exclusiones

↩ detalle — "ISA"
ISA:

IDE: @github en chat no respeta exclusiones en VS Code/VS.

Semántica: tipos/símbolos de archivo excluido pueden filtrarse por el IDE.

Ámbito: solo aplica a miembros de la org configurada.

Solución de problemas comunes — mnemónico: "FES: Faltan, Exclusiones, Sugerencias"
F — Faltan sugerencias

Copilot no proporciona autocompletado

↩ pasos

1. Verifica conexión a Internet.

2. Actualiza la extensión de Copilot.

3. Verifica compatibilidad del IDE.

4. Revisa las exclusiones de contenido activas.

E — Exclusiones no funcionan

Las exclusiones no surten efecto

↩ causas

Demora: Los cambios tardan hasta 30 min. Recarga la configuración manualmente.

Ámbito: Solo aplica a miembros de la org configurada.

IDE: Verifica ícono de Copilot en barra de estado (línea diagonal = excluido).

S — Sugerencias insatisfactorias

La calidad de las sugerencias es baja

↩ técnicas

1. Contexto claro: usa comentarios descriptivos y nombres de variables significativos.

2. Ctrl+Enter en VS Code para desencadenar Copilot.

3. Indicación más larga: experimenta con distintas longitudes.

Resumen

4 planes: Gratis/Pro (básico, solo filtro público) → Business (admin, IP, exclusiones) → Enterprise (personalización avanzada, bases de conocimiento).
Protecciones IDC: Indemnización IP (bloquear código público como requisito), DPA y Centro de confianza.
Exclusiones CIO: Configurar en repo/org, el impacto es triple (no completado, no contexto, no chat), con límites ISA (IDE, Semántica, Ámbito).
Problemas FES: Faltan sugerencias (internet/extensión/IDE/exclusiones), Exclusiones no aplican (demora 30min, recarga), Sugerencias pobres (contexto, indicación más larga).
