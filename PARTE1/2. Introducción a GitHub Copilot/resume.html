6 características principales · nemotecnia: "Chat Resumido Ayuda Con Spaces Autónomo"

C — Copilot Chat↻ clic
Chat interactivo en el IDEFormular preguntas sobre código, obtener explicaciones, generar pruebas o documentación, explorar nuevas características. Entiende el contexto del proyecto.
R — Resúmenes de PR↻ clic
Descripción automáticaAl abrir un PR, Copilot genera automáticamente un resumen de los cambios. Reduce el trabajo al escribir PRs y mejora la claridad en revisiones.
A — Asistencia de revisión↻ clic
Ayuda en code reviewSugiere problemas potenciales, describe cambios, señala casos límite y propone mejoras. Acelera ciclos de revisión y reduce sobrecarga manual.
C — Copilot CLI↻ clic
Terminal inteligenteSugerencias de comandos, generación de scripts de shell, explicación de errores y resultados, generación y mejora de proyectos desde la CLI.
S — Copilot Spaces↻ clic
Espacio colaborativo con contextoLugar dedicado para explorar estructura del proyecto, preguntas de planeación, refinar requisitos, iterar diseños. Copilot mantiene el conocimiento del repo.
A — Agente en la nube↻ clic
Tareas autónomas multi-pasoGenera múltiples archivos relacionados, implementa un conjunto de características, construye andamiaje desde una especificación. Opera en tu dirección.

Planes de suscripción · nemotecnia: "Fantásticos Programadores Pueden Brillar Enormemente"

F — Free · Gratis↻ clic
Para individuos sin costoFinalizaciones de código + chat con límite mensual. Acceso a modelos de IA avanzados. Inicio: ícono Copilot en VS Code → "Iniciar sesión gratis".
P — Pro · Individual↻ clic
Más allá del FreeLímites más altos, acceso prioritario a modelos recientes, sugerencias y explicaciones avanzadas. Integración con VS Code, Visual Studio, JetBrains, Neovim.
P — Pro+ · Premium↻ clic
Todo Pro + capacidad extraCapacidad adicional de solicitudes premium y acceso prioritario a infraestructura. Para usuarios con mayores necesidades de uso y alta demanda.
B — Business · Org↻ clic
Gestión centralizadaControles de directivas, filtrado de vulnerabilidades, referencia de código público, indemnización por IP, seguridad y privacidad de nivel empresarial.
E — Enterprise · Grande↻ clic
Todo Business + personalizaciónSugerencias con código interno, indexación del codebase, integración con GitHub Enterprise Cloud, ajuste de modelos para toda la organización.

Formas de interacción · nemotecnia: "Inline Chat Con Paleta Comenta"

I — Inline · Tab/→↻ clic
Autocompletado en tiempo realCopilot analiza contexto y sugiere código como texto atenuado. Tab o → para aceptar. Esc o seguir escribiendo para rechazar.
C — Chat de Copilot↻ clic
Panel conversacionalAbre el panel lateral en el IDE. Preguntas en lenguaje natural. Ideal para explorar conceptos, obtener explicaciones o generar fragmentos complejos.
C — Chat insertado · Ctrl+I↻ clic
Contextual en el cursorCtrl+I (Win/Linux) / Cmd+I (Mac). Conversación específica del punto del cursor. Usa /comandos para mayor eficiencia sin cambiar contexto.
P — Paleta · Ctrl+⇧+P↻ clic
Acceso rápido a accionesCtrl+Shift+P → escribe "Copilot". Accede a: Explica esto, Genera pruebas unitarias, y más acciones directas sin usar el ratón.
C — Comentario → Código↻ clic
Lenguaje natural → funciónEscribe un comentario describiendo la función deseada. Al presionar Enter, Copilot genera el cuerpo completo basado en la descripción.
Alt+] — Alternativas↻ clic
Explorar opcionesAlt+] (Win/Linux) / Option+] (Mac) para recorrer alternativas. El ícono de bombilla indica que hay múltiples sugerencias disponibles.

Comandos de barra diagonal · nemotecnia: "Esta Solución Tiene Código"

E — /explain↻ clic
Explicar código seleccionadoProporciona explicación detallada del bloque seleccionado. Útil para aprender lógica existente o revisar código de otros miembros del equipo.
S — /suggest↻ clic
Sugerir mejorasOfrece sugerencias de código basadas en el contexto actual del archivo y la función. Propone alternativas y optimizaciones posibles.
T — /tests↻ clic
Generar pruebas unitariasGenera casos de prueba para la función o clase seleccionada. Ahorra tiempo en testing y ayuda a detectar errores al principio del desarrollo.
C — /comment↻ clic
Comentario → fragmento de códigoConvierte comentarios descriptivos en código funcional. Describe la funcionalidad en lenguaje natural y Copilot genera la implementación.

Configuración y solución de problemas · VS Code

Instalar extensión↻ clic
Visual Studio MarketplaceBuscar "GitHub Copilot" → Instalar → Abrir VS Code → Iniciar sesión con cuenta GitHub. Habilitar/deshabilitar desde el ícono de estado en la barra inferior.
Solución de problemas↻ clic
Logs y diagnósticosLogs: Ctrl+Shift+P → "Desarrollador: Abrir el archivo de registro". Red: Ctrl+Shift+P → "GitHub Copilot: Recopilar diagnósticos". Ayuda → Herramientas de desarrollo para logs de Electron.

Resumen del módulo

GitHub Copilot es un programador de pares con IA que transforma el ciclo de vida del desarrollo: desde la escritura de código hasta PRs, revisiones y CLI.

6 características clave: Chat · Resúmenes de PR · Asistencia de revisión · CLI · Spaces · Agente en la nube.

5 planes: Free · Pro · Pro+ · Business · Enterprise — cada uno adaptado a individuos, equipos o grandes organizaciones.

Copilot aprende del contexto: mantener el código bien estructurado y comentado produce sugerencias más precisas. Cuanto más interactúas, mejor comprende tu estilo.
