  • Diseñar prompts efectivos que generen sugerencias precisas y útiles
  • Aplicar GitHub Copilot para mejorar proyectos JavaScript existentes

GitHub Copilot con JavaScript

Asistente de IA que completa código y responde preguntas — desde una sugerencia de una línea hasta funciones completas, guiado por tus comentarios y contexto abierto.

Cómo funciona Copilot

Motor

OpenAI Codex

Modelo entrenado en código público y lenguaje natural. Analiza el archivo activo y los archivos relacionados para derivar contexto y generar sugerencias coherentes.

Sugerencia fantasma

Texto gris en el editor

Copilot muestra la sugerencia como texto gris sobre el cursor. Presiona Tab para aceptar. Si hay varias opciones, usa Ctrl+Enter para ver el panel completo.

Copilot Chat

Preguntas en lenguaje natural

Interfaz de chat interactiva para formular preguntas sobre código, recibir explicaciones, generar tests y refactorizar — sin salir del IDE.

Formas de solicitar — mnemónico CMC

Comentario · Markdown · Chat

C — Comentario en código

// descripción de la tarea

Escribe un comentario en un archivo .js, .py, etc. y Copilot completará el bloque siguiente. Ejemplo:

// Create an API endpoint using
// React that accepts JSON POST

M — Markdown

Escribe texto y espera

En un archivo .md, escribe el texto descriptivo y espera unos segundos. Copilot responde con sugerencias de código o documentación dentro del mismo archivo.

C — Chat interactivo

Panel lateral de preguntas

Abre GitHub Copilot Chat desde la barra lateral. Puedes pasar contexto de otros archivos abriéndolos antes de preguntar — Copilot los usa como contexto adicional.

Ingeniería rápida — prompts de calidad

Prompt vago

Resultado impredecible

// Create an API endpoint

Ambiguo: no especifica framework, método HTTP, tipo de datos ni propósito. Copilot puede generar algo completamente diferente a lo que necesitas.

Prompt específico

Resultado enfocado

// Create an API endpoint using
// React that accepts a JSON
// payload in a POST request

Claro: indica framework, método y formato. Copilot genera exactamente lo que el equipo necesita.

Procedimientos recomendados — mnemónico SRRC

Simple → Recorre → Reescribe → Contexto

S — Simple primero

Empieza con un prompt básico:
"create an HTML form with a text field and button"
Luego añade complejidad:
"Add event listener to POST to /generate and show result in #result"

R — Recorre opciones

Usa Ctrl+Enter (o Cmd+Enter en Mac) para abrir el panel de sugerencias completo y elegir la más adecuada entre las opciones que Copilot genera.

R — Reescribe si falla

Si no obtienes los resultados deseados: reescribe el mensaje con más detalles o empieza a escribir el código tú mismo para que Copilot lo complete automáticamente.

C — Contexto abierto

Copilot usa los archivos abiertos en el editor como contexto adicional. Abre los archivos relacionados antes de escribir tu prompt para obtener sugerencias más precisas.

Configuración y plan gratuito

Cuenta GitHub

Requisito base

Necesitas una cuenta de GitHub activa. Copilot es un servicio de GitHub — sin cuenta no hay acceso. El plan Copilot Free no requiere tarjeta de crédito.

Plan Free

2 000 completados · 50 chats/mes

2 000 autocompletados de código y 50 mensajes de chat al mes. Suficiente para aprender. Educadores, estudiantes y OSS maintainers pueden obtener Copilot Pro gratis.

Extensión VS Code

GitHub Copilot + Chat

Instala GitHub Copilot y GitHub Copilot Chat desde el marketplace. También disponible para JetBrains, VIM, XCode y Visual Studio.

GitHub Codespaces

Entorno en la nube

VS Code preconfigurado

Entorno de desarrollador hospedado en la nube con extensión Copilot preinstalada. Incluye dependencias, bibliotecas y configuraciones listas para usar. 60 horas gratis/mes con 2 cores.

Puerto 1234

npm start dentro del Codespace

La aplicación de cartera JavaScript corre en el puerto 1234 del Codespace. El terminal muestra el mensaje de servidor activo una vez completada la instalación.

Ejercicio: Portfolio JS — mnemónico PAS

Personaliza · Anima · Supervisa

P — Personaliza siteProps

src/App.jsx

const siteProps = {
  name: "Tu Nombre",
  title: "Web Developer",
  email: "tu@email.com",
  gitHub: "usuario",
  linkedIn: "perfil",
  twitter: "handle",
  youTube: "canal",
};

A — Anima con Copilot

src/styles.css

/* add an amazing animation
   to the social icons */

img.socialIcon:hover {
  animation: bounce 0.5s;
  animation-iteration-count:
    infinite;
}
@keyframes bounce {
  0%   { transform: scale(1); }
  50%  { transform: scale(1.2); }
  100% { transform: scale(1); }
}

S — Supervisa en vivo

Hot-reload automático

El Codespace recarga la página automáticamente al guardar. Pasa el cursor sobre los iconos del pie de página para ver la animación bounce en acción. Confirma cambios antes de eliminar el Codespace.

Resumen — GitHub Copilot (motor: OpenAI Codex) actúa como copiloto de IA dentro del IDE: lee el contexto de los archivos abiertos y ofrece sugerencias en gris que se aceptan con Tab. La calidad de las sugerencias depende directamente de la calidad del prompt — aplica el patrón SRRC (Simple → Recorre → Reescribe → Contexto). En el ejercicio de portfolio JavaScript, usaste un comentario CSS para que Copilot generara una animación bounce completa, demostrando que la IA puede asistir en código, estilos, documentación y pruebas.
