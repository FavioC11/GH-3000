GitHub Copilot con Python

Usa indicaciones en lenguaje natural para que Copilot genere modelos Pydantic, endpoints FastAPI e importaciones — acelerando el ciclo completo de desarrollo de APIs web.

Cómo funciona con Python

Motor

OpenAI Codex

Analiza el archivo activo y los archivos relacionados para derivar contexto. Genera sugerencias de nuevas líneas o funciones completas en Python basándose en comentarios y código existente.

Aceptar sugerencias

Tab · Ctrl+Enter

La sugerencia aparece como texto gris en el editor. Presiona Tab para aceptar. Usa Ctrl+Enter para ver el panel completo con múltiples opciones y elegir la más adecuada.

Ingeniería de solicitudes — prompts Python

Indicación vaga

Resultado impredecible

# Create an API endpoint

Ambigua: sin framework, sin método HTTP, sin tipo de datos. Copilot puede sugerir un framework desconocido o un endpoint que requiere datos inesperados.

Indicación específica

Resultado enfocado

# Create an API endpoint
# using the FastAPI framework
# that accepts a JSON payload
# in a POST request

Clara: especifica framework, método y formato. Copilot genera exactamente lo necesario para el proyecto.

Procedimientos recomendados — mnemónico SRRC

Simple → Recorre → Reescribe → Contexto

S — Simple primero

Empieza con un prompt básico y añade complejidad progresivamente. Primero el formulario, luego el event listener, luego el endpoint — paso a paso.

R — Recorre opciones

Usa Ctrl+Enter (Mac: Cmd+Enter) para abrir el panel de sugerencias y seleccionar la más adecuada entre todas las que genera Copilot.

R — Reescribe si falla

Si la sugerencia no es la esperada, reformula la indicación con más detalles o empieza a escribir el código tú mismo para que Copilot lo complete automáticamente.

C — Contexto abierto

Copilot usa los archivos abiertos como contexto adicional. Abre los archivos relacionados o referencialos en Copilot Chat para obtener sugerencias más coherentes con el proyecto.

Codespaces con Python — puerto 8000

Codespace Python

uvicorn · puerto 8000

Codespaces instala todas las extensiones y dependencias en el contenedor. Ejecuta uvicorn automáticamente para iniciar la API web. El terminal confirma el servidor en el puerto 8000.

FastAPI autodocs

/docs interactivo

FastAPI genera automáticamente documentación interactiva en /docs. Usa el botón POST → Probarlo → Ejecutar en el Explorador simple para probar cada endpoint sin cliente externo.

Token pseudoaleatorio

Caso de uso base

La API de plantilla genera un token pseudoaleatorio — habitual en rutinas de identificación. El ejercicio la extiende con un nuevo endpoint que acepta texto y retorna un checksum.

Ejercicio: Extender la API Python — mnemónico MEI

Modelo Pydantic · Endpoint POST · Importaciones

M — Modelo Pydantic

Paso 1 · main.py

Agrega un comentario en main.py para que Copilot genere el modelo automáticamente:

class Text(BaseModel):
    text: str

Pydantic valida el cuerpo JSON de la solicitud POST.

E — Endpoint POST

Paso 2 · Checksum de texto

Indicación precisa para Copilot:

# Create a FastAPI endpoint
# that accepts a POST request
# with a JSON body containing
# a single field called "text"
# and returns a checksum
# of the text

I — Importaciones

Paso 3 · Copilot Chat

El código generado puede fallar si faltan base64 y os. Usa Copilot Chat para pedir las importaciones faltantes, o agrégalas manualmente:

import base64
import os
Formas de indicar a Copilot

Comentario en .py

Indicación directa en código

Escribe un comentario # en el archivo .py y Copilot genera el bloque siguiente. Es la forma más natural para Python — el comentario queda como documentación del intento.

Texto en Markdown

Espera unos segundos

En un archivo .md, escribe la descripción y espera. Copilot responde con sugerencias de código o documentación dentro del mismo archivo markdown.

Copilot Chat

Preguntas + archivos de referencia

Interfaz conversacional para preguntar, refinar y corregir código. Puedes referenciar otros archivos del proyecto para que Copilot tenga contexto completo al generar la respuesta.

Resumen — GitHub Copilot con Python sigue el mismo modelo que con JavaScript: indicaciones en lenguaje natural → sugerencias en gris → Tab para aceptar. La clave está en la calidad del prompt (patrón SRRC). En el ejercicio extendiste una API web FastAPI en tres pasos MEI: generaste un modelo Pydantic con un comentario, creaste un endpoint POST de checksum con una indicación específica, y usaste Copilot Chat para detectar y agregar las importaciones faltantes (base64, os). Verifica el resultado en /docs con la UI interactiva de FastAPI.
